# 나의 사역 스타일 테스트

Vite + React 기반 웹앱 초안입니다.

## 실행 방법

```bash
npm install
npm run dev
```

## 배포

Vercel에 올릴 때는 프로젝트 폴더를 연결하면 됩니다.

- Framework Preset: Vite
- Build Command: npm run build
- Output Directory: dist

## 수정할 곳

- 신청 링크: `src/App.jsx`의 `APPLY_URL`
- 질문/결과 문구: `src/data.js`
- 디자인: `src/styles.css`
